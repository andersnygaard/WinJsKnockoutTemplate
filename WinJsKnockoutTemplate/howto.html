Howto

Out of the box, Windows 8 projects uses WinJs. It's ok, but there's nothing stopping you from using your favorite frameworks. Focus on the creative part. Use this app as an ispiration.

Save as template

In Visual Studio 2012, go to File-Export Template and finish the wizard. This app is now saved as a template project. The next time you start a Visual Studio project, you should find the WinJsKnockoutTemplate under JavaScript projects.
